# YAML/MD Editor

Willkommen im YAML/MD Editor!

## Funktionen

- **YAML-Dateien bearbeiten** mit Syntax-Highlighting
- **Markdown-Dateien bearbeiten** mit Syntax-Highlighting
- **Dateiverwaltung** (erstellen, bearbeiten, löschen)
- **Benutzerverwaltung** für Administratoren

## Tastenkombinationen

| Kombination | Aktion |
|-------------|--------|
| `Ctrl + S` | Datei speichern |

## Unterstützte Dateiformate

1. `.yaml` / `.yml` - YAML-Dateien
2. `.md` / `.markdown` - Markdown-Dateien
